# 🧠 Learning to Plan

Train LLMs for classical planning using test-time compute.

---

## 🔧 Development Commands

### 🐍 Conda

```bash
conda env create -n learning_to_plan -f environment.yml
conda activate learning_to_plan
pip install -e .
```


### 🐳 Docker

```bash
make build
make run
make shell
make dev-run
make clean
```
